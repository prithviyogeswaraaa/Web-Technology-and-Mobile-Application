Project Title

Web Page Creation using HTML – Chennai Map with Hotspots, Frames & Links

Description

This project demonstrates the creation of a web page using HTML that embeds a map of Chennai with clickable hotspots, frames (iframe), and navigation links. The webpage is styled using three types of CSS:

External CSS

Embedded CSS

Inline CSS

The project helps in understanding how to integrate multimedia elements like maps, create interactive hotspots, and organize content using frames.

Features

Embedded Chennai city map image

Clickable hotspots for:

Marina Beach

Fort St. George

Kapaleeshwarar Temple

Embedded Google Map using iframe

Internal and external navigation links

Styling using:

External CSS (style.css)

Embedded CSS (<style> tag)

Inline CSS (style attribute)

Files Included

index.html – Main HTML file

style.css – External CSS file

chennai_map.jpg – Image used for hotspot mapping

Technologies Used

HTML5

CSS3

How to Run

Download all project files into the same folder.

Open index.html in any web browser (Chrome, Edge, Firefox, etc.).

Click on the hotspots to navigate to information pages.

Check the embedded map and links for proper functionality.

Result

The HTML page is successfully created with an embedded Chennai map along with hotspots, frames (iframe), and links. All CSS styling techniques are properly implemented and tested.
