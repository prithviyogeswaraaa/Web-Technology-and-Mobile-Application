# Web Page Creation using HTML  
## Chennai Map with Hotspots, Frames & Links  

---

## Description  

This project demonstrates creating a web page that embeds a Chennai city map with clickable hotspots, iframe (frames), and navigation links.

The page uses:
- External CSS  
- Embedded CSS  
- Inline CSS  

---

## Features  

- Embedded Chennai city map image  
- Clickable hotspots  
- Embedded Google Map using iframe  
- Internal and external navigation links  
- Styled using three types of CSS  

---

## Files Included  

- index.html  
- style.css  
- chennai_map.jpg  

---

## Technologies Used  

- HTML5  
- CSS3  

---

## Result  

The HTML page was successfully created with embedded Chennai map, hotspots, iframe, and links using external, embedded, and inline CSS.
